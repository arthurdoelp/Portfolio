Party in my Pocket

Description

This app uses the CocktailDB to gather user inputs of cocktail ingredients to related cocktails containing the ingredients. It also uses the Spotify API to create curated playlists for each of the cocktails to set the party mood of the cocktail. There is also the Giphy API to display a related Gif as well as the Wikipedia API to share trivia information witht the user. There is also a "save cocktail" feature to save cocktails that the user likes to the browser local storage.
